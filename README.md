# FAQ Chatbot with NLP

A domain-specific chatbot that answers frequently asked questions using a pre-trained BERT model.

## What I used
- Python
- HuggingFace Transformers (BERT)
- Flask
- JSON

## What I did
- Used a pre-trained BERT model from HuggingFace for intent matching
- Built a REST API using Flask to handle chatbot requests
- Tested the chatbot with different question variations

## What I learned
This was my first time working with HuggingFace and deploying a model as an API. 
I learned how pre-trained models can be used without training from scratch.
